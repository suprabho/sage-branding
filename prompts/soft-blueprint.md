## Logo Concepts

**Primary Wordmark:**
"SAGE" set in Space Mono, weight 700, all-uppercase. Tight letterspacing (-1%). A small Electric Blue (#2563EB) underline sits 4px below the baseline — like a code editor cursor or terminal prompt. Color: Midnight (#0F172A) on white, or reversed on Electric Blue.

**Icon Mark:**
A stylized brain/circuit node — a circle with 3 short radiating lines (like a simplified neural network node). Clean 2px stroke, Electric Blue. Geometric, precise. Optionally: the node sits inside a rounded square (8px radius) with a Lemon (#FFD93D) fill background.

**Logo Animation:**
The wordmark types itself in, character by character, like a terminal input — 60ms per character, with a blinking cursor that remains for 400ms after completion then fades. The icon mark draws its strokes sequentially (circle → lines) over 500ms with ease-out timing. Feels like booting up — precise, engineered.

## Patterns

**Dotted Grid:**
Cool Slate (#CBD5E8) dots at 10% opacity on Ice White, radial-gradient circles 1px, spaced 20px. Signature pattern — use on chat backgrounds, landing page sections, social posts. Blueprint aesthetic.

**Concentric Arcs:**
Quarter-circle arcs radiating from corner, Electric Blue at 8% opacity. Stroke-only, 1px. Use as decorative element behind profile sections or hero areas. Technical, precise.

**Marker-Pen Highlight:**
Lemon (#FFD93D) rectangle behind key text phrases. Slightly rough edges (2px border-radius, not perfectly sharp). Signature text treatment across all touchpoints — chat, email, social, web.

**Grid Overlay:**
Thin Midnight lines at 3% opacity forming a 60px engineering grid. Use as background on data-heavy sections or dashboards. Reinforces "we engineered this."

## Icons

**Style Direction:**
Phosphor Icons, regular variant with 2px stroke. Electric Blue (#2563EB) default. Clean corners rounded at 2px. Never filled by default — fill only on active/selected states. Consistent 2px weight across all sizes.

**Custom Icon Set (if extending beyond Phosphor):**
24px grid, 2px stroke, round caps. Geometric precision — circles are perfect, lines are straight. No organic curves. Suggested custom icons: Data chart, Milestone flag, Blueprint document, Timer, Checklist. Each icon fits a 20×20px optical area within the 24px frame.

## Imagery

**Photography Direction:**
High-contrast, slightly cool-toned. Parents as capable, in-control adults — not soft-focus emotional moments. Think: a parent confidently managing a routine, organizing a space, making a decision.

- Avoid: overly warm/dreamy, soft focus, pastel grading
- Favor: clean environments, good lighting, parents with agency. Natural but composed.
- Color grade: slightly desaturated, cool midtones, lifted shadows. Matte, editorial.
- Crop: tight, purposeful. Subjects centered or rule-of-thirds. Minimal negative space — information-dense framing.

**Illustration Direction (if used):**
Technical, diagrammatic. Think: infographics, flowcharts, simple data visualizations. Electric Blue primary, Lemon accents, Cool Slate for secondary elements. Clean vector lines, no hand-drawn feel. Isometric perspectives for objects. Never cute or whimsical.

**Image Treatments:**
- Sharp corners: 8px radius on all containers
- Electric Blue 2px border around image containers
- Lemon offset shadow: 4px 4px 0px on featured images
- No overlay filters — images stand as-is with strong native contrast
